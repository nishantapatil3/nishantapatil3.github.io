# [Nishant's Portfolio](https://nishantapatil3.github.io/)

## Local Development

```bash
npm install
npm run dev
```

Then open: http://localhost:5173

Posts index: http://localhost:5173/posts/

## Build

```bash
npm run build
npm run preview
```
